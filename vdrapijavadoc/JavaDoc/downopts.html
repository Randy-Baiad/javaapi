(1) Formats that can use policy information can be converted to CSV, SLYK, or XML.

Notes:

  • Append to Stream / File - Where multiple pages are being downloaded, indicates whether the format supports creation of a single file.
  • Scale - Indicates whether the output can be re-sized.
  • Rotate - Indicates whether an angle of rotation can be specified for the output.
  • Grayscale - Indicates whether the output can be displayed as a grayscale image.
  • Quality Factor / Passes - Can be set when AFP, PCL, PDF or Xerox is being converted to JPEG.
  • Sub pages - Can be set when a multi-page AFP is being converted to any of the supported image types.
  • DPI - Indicates whether the output resolution can be specified.
